# JobAspirations 🚀

**JobAspirations** is a full-stack job portal built to simulate real-world hiring workflows between **students, recruiters, and admins**.

This is not a demo project — it is a **production-style system** with role-based access, approval flows, and deployed infrastructure.

🌐 Live: https://jobaspirations.in

---

## ✨ Key Features

### 👩‍🎓 Student
- Authentication & profile management
- Browse and search jobs
- Save jobs
- Apply for jobs
- Track application status

### 🧑‍💼 Recruiter
- Recruiter registration
- Admin approval required before activation
- Company profile management
- Post and manage job listings
- View applicants
- Update application statuses

### 🛡️ Admin
- Approve / reject recruiter requests
- Control platform access
- Maintain platform integrity

---

## 🧠 System Design Highlights

- Role-based access control (Student / Recruiter / Admin)
- Centralized authentication middleware
- Feature-based backend architecture
- Real-world approval and status workflows
- Production deployment with reverse proxy
